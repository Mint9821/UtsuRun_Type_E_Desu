# UtsuRun_Type_E_Desu
本リポジトリは「写〇ンです」のレンズを、S〇ny Eマウントに変換するモデルです。 </br>
モデルを改変する場合は、電脳妖精☆ミントまでご一報いただければ幸いです。
こちらは任意です。 </br>
一応、MIT licenseを置いておきます。

> 可能でしたら、ぜひモデルを公開していただけるともっと嬉しいです。

## 推奨印刷設定
- フィラメント：PLAなど、基本的になんでも大丈夫かと思います。
- 積層ピッチ：0.12 mm
- 射出幅：0.4 mm
- 底面の設定：ご自分の環境を加味して設定してください。

## 組み立て方
1. 「写ル〇です」のレンズのみを取り出します。
2. 画像の通りに組み立てます。
